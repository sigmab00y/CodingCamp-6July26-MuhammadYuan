Fitur & Isi di Dalam Project

1. Sistem Multi-Akun Lokal: Menyediakan form Login dan Register aman di halaman lobi utama yang mengisolasi data transaksi 
spesifik per username.

2. Manajemen Transaksi: Memungkinkan pencatatan pengeluaran secara real-time, pembuatan kategori kustom secara dinamis, 
serta fitur penghapusan data.

3. Kontrol Anggaran & Sortir: Dilengkapi fitur penetapan batas anggaran bulanan untuk mendeteksi pengeluaran berlebih 
(overbudget) serta fitur sortir riwayat transaksi (Terbaru, Nominal, Kategori).

4. Visualisasi Data (Chart.js): Mengintegrasikan grafik diagram lingkaran (pie chart) interaktif untuk memetakan persentase 
distribusi alokasi dana secara visual.

5. Asisten Virtual & UX: Menghadirkan maskot interaktif yang memberikan rangkuman analisis keuangan dan tautan edukatif, 
serta fitur dark mode untuk kenyamanan mata.

6. Penyimpanan Data
Seluruh data akun dan riwayat transaksi disimpan secara instan dan privat di sisi klien menggunakan Web Storage AP 
(LocalStorage), sehingga aplikasi dapat berjalan responsif tanpa memerlukan database server eksternal.
